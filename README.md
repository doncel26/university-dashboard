# University Admissions Dashboard

This repository contains a Streamlit dashboard and a Colab notebook for analyzing university admissions, enrollment, retention and satisfaction.

## Files
- `app.py` : Streamlit dashboard
- `university_student_data.csv` : dataset (place it in the repo root)
- `requirements.txt` : Python dependencies
- `Data_Visualization_and_Dashboard_Deployment.ipynb` : Colab notebook with EDA and plots

## How to run locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Deploy to Streamlit Cloud
1. Push this repo to GitHub.
2. On https://streamlit.io/cloud, click "New app" and select your GitHub repo.
3. Set the main file to `app.py`.
4. Deploy.
